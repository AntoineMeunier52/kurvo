# Kurvo

Open-source visual page builder for Vue.

## Setup

```bash
pnpm install
pnpm dev
```

## Scripts

```bash
pnpm build       # build all packages
pnpm test        # run tests
pnpm lint        # lint
pnpm typecheck   # typecheck
```

## Packages

| Package | Description |
|---|---|
| `@kurvo/core` | Types, state, validation — zero Vue/DOM deps |
| `@kurvo/vue` | `<KurvoEditor />` component + `<DocumentRenderer>` |
